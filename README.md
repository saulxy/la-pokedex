# la-pokedex
Cross-platform Pokedex (Linux, Windows and macOS)

# Setting up your workspace
Assuming you have npm, install the dependencies:
```
npm install --save-exact
```
Running the app (dev mode):
```
.\node_modules\.bin\electron .
```
